إضافة التصنيفات الأكاديمية لـ Tutor LMS - جامعات، مدارس، وكورسات عامة

== Description ==

إضافة متكاملة لـ Tutor LMS تتيح إنشاء نظام تصنيف أكاديمي متكامل يشمل:

* **الجامعات والكليات**: نظام هرمي للجامعات → الكليات → الأقسام
* **المدارس**: نظام للمدارس والمراحل الدراسية
* **الكورسات العامة**: كورسات متاحة لجميع المستخدمين

== Features ==

* لوحة تحكم متكاملة مع Tutor LMS
* حقول تسجيل مخصصة للطلاب والمعلمين
* فلترة ذكية للكورسات حسب التخصص الأكاديمي
* دعم كامل للغة العربية
* واجهة مستخدم متجاوبة
* تكامل سلس مع Tutor LMS

== Installation ==

1. قم برفع مجلد `tutor-lms-academic-pro` إلى مجلد `/wp-content/plugins/`
2. قم بتفعيل الإضافة من خلال قائمة 'الإضافات' في ووردبريس
3. تأكد من تفعيل إضافة Tutor LMS أولاً
4. انتقل إلى Tutor LMS → التصنيفات الأكاديمية لبدء الإعداد
